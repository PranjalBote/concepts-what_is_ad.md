# What is Active Directory?

Active Directory (AD) is a directory service developed by Microsoft that helps organizations manage and organize users, computers, and resources within a network.

It acts like a centralized database that stores information about all objects in a network and allows administrators to control access and permissions.

## Key Functions
- Authentication (Who are you?)
- Authorization (What can you access?)
- Centralized management of users and devices

## Example
When you log into a company computer using a username and password, Active Directory verifies your identity.

## Why AD is Important in Cybersecurity
- Central point of attack for attackers
- Misconfigurations can lead to privilege escalation
- Used in most enterprise environments

## Components Used
- Domain Controller
- LDAP
- Kerberos
- DNS
